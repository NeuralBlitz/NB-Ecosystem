9.  **Clock Speed and Clock Cycles:** The "heartbeat" of a computer that synchronizes operations.

### II. Central Processing Unit (CPU) Architecture (The Brain)
11. **The Fetch-Decode-Execute Cycle:** The basic operational cycle of a CPU.
12. **Arithmetic Logic Unit (ALU):** The part of the CPU that performs arithmetic and logical operations.
13. **Control Unit (CU):** The part of the CPU that directs the flow of operations.
14. **CPU Registers:** The small, extremely fast memory locations within the CPU itself.
15. **Pipelining:** A technique where multiple instructions are overlapped in execution, like an assembly line.
16. **Multi-core Processors:** Placing two or more independent processing cores on a single chip.
17. **Superscalar Architecture:** Processors that can execute more than one instruction during a single clock cycle.
18. **CPU Caches (L1, L2, L3):** The hierarchy of small, fast memory used to reduce the latency of accessing main memory.
19. **Branch Prediction:** A technique used by modern CPUs to guess the outcome of a conditional operation to improve performance.
20. **System on a Chip (SoC):** An integrated circuit that combines many components of a computer (CPU, GPU, memory) onto a single chip.

21. **Random Access Memory (RAM):** The main volatile memory used by a computer (DRAM vs. SRAM).

23. **Virtual Memory and Paging:** Using secondary storage to simulate extra main memory.
24. **Read-Only Memory (ROM) & Firmware (BIOS/UEFI):** Non-volatile memory that holds the initial boot instructions.
25. **Hard Disk Drives (HDDs):** Traditional magnetic spinning-platter storage.
26. **Solid-State Drives (SSDs):** Storage devices that use flash memory (NAND).
27. **Flash Memory:** The non-volatile storage technology used in SSDs, USB drives, and SD cards.
28. **RAID (Redundant Array of Independent Disks):** Combining multiple disk drives into one logical unit for performance or redundancy.
29. **Error-Correcting Code (ECC) Memory:** A type of RAM that can detect and correct common kinds of data corruption, used in servers.
30. **Memory Controllers:** The digital circuit that manages the flow of data going to and from the main memory.

32. **Chipsets:** The logic on the motherboard that manages data flow between the CPU, memory, and peripherals.

34. **PCI Express (PCIe):** The high-speed standard interface for connecting components like GPUs and SSDs.
35. **Universal Serial Bus (USB):** The industry standard for connecting and powering peripherals.
36. **SATA & NVMe:** The primary interfaces for connecting storage drives to the motherboard.
37. **Voltage Regulator Modules (VRMs):** Components on the motherboard that provide clean and stable power to the CPU.
38. **Expansion Slots:** Sockets on the motherboard for adding expansion cards.
39. **Power Connectors:** The various connectors used to supply power to the motherboard and components.
40. **Front Panel Connectors:** The pins used to connect the computer case's power buttons, LEDs, and USB ports.
